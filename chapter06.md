# 6장 FASTA 파일과 파일 입출력의 기초

지금까지는 DNA 서열을 코드 안에 직접 입력해 실습했습니다. 그러나 실제 생물정보학에서는 서열 데이터를 직접 손으로 타이핑하기보다, 외부 파일에서 불러오는 경우가 훨씬 많습니다. 유전체 데이터, 단백질 서열, 유전자 발현 정보는 대부분 파일 형태로 저장되며, 분석가는 필요한 정보를 파일에서 읽어 와야 합니다.

그중에서도 FASTA는 생물정보학에서 가장 널리 사용되는 서열 파일 형식 중 하나입니다. FASTA 파일을 이해하면, 인터넷에서 내려받은 실제 생물학 데이터를 파이썬으로 읽고 분석하는 첫걸음을 뗄 수 있습니다. 이 장에서는 파일 입출력의 기본을 익히고, FASTA 파일에서 순수한 서열 정보만 추출하는 방법을 배웁니다.

그림 6-1은 이 장의 핵심을 보여 줍니다. FASTA 파일에는 설명 줄과 실제 서열 줄이 함께 들어 있으며, 우리는 파이썬으로 서열 부분만 골라 분석에 사용할 수 있습니다.

## 학습 목표

- 파일 입출력이 무엇인지 설명할 수 있다.
- `open()` 함수를 사용해 텍스트 파일을 읽고 쓸 수 있다.
- FASTA 형식의 기본 구조를 이해할 수 있다.
- FASTA 파일에서 서열 정보만 추출하는 코드를 작성할 수 있다.

## 파일은 왜 필요한가?

프로그램 안에 직접 데이터를 입력하는 방식은 작은 예제에서는 편리하지만, 실제 연구 데이터에는 적합하지 않습니다. 서열이 수백 개, 수천 개가 되면 파일 없이 데이터를 관리하기가 거의 불가능합니다.

파일은 데이터를 저장하고 다시 불러오는 기본 수단입니다. 프로그램이 종료된 뒤에도 데이터가 사라지지 않고, 다른 사람과 공유하거나 다시 분석할 수 있다는 점에서 매우 중요합니다. 생물정보학에서 파일 입출력은 실험 결과를 다루는 기본 기술입니다.

## `open()`으로 텍스트 파일 읽고 쓰기

파이썬에서 파일을 다룰 때는 `open()` 함수를 사용합니다. 예를 들어 텍스트 파일에 문장을 저장하려면 다음과 같이 쓸 수 있습니다.

```python
with open("memo.txt", "w") as f:
    f.write("I love Bioinformatics!\n")
    f.write("Python is powerful.\n")
```

여기서 `"w"`는 쓰기(write) 모드입니다. 파일에 내용을 기록합니다.

파일을 읽을 때는 `"r"` 모드를 사용합니다.

```python
with open("memo.txt", "r") as f:
    content = f.read()

print(content)
```

이렇게 하면 저장된 내용을 다시 불러와 출력할 수 있습니다.

## FASTA 형식 이해하기

FASTA 파일은 보통 다음과 같은 형태를 가집니다.

```text
>Human_insulin
ATGCCCTCCAGGACAGG...
>Hemoglobin_alpha
ATGGTGCTGTCTCCTG...
```

`>`로 시작하는 줄은 헤더(header)입니다. 이 줄에는 서열의 이름이나 설명이 들어갑니다. 그 아래 줄에는 실제 DNA 또는 단백질 서열이 기록됩니다.

즉, FASTA 파일을 읽을 때 중요한 점은 다음 두 가지입니다.

1. `>`로 시작하는 줄은 설명 정보다.
2. 나머지 줄은 실제 분석할 서열 데이터다.

## 헤더와 서열 구분하기

파이썬에서는 파일을 한 줄씩 읽으면서, `>`로 시작하는 줄을 건너뛰는 방식으로 서열만 추출할 수 있습니다.

```python
with open("sample.fasta", "r") as f:
    for line in f:
        if line.startswith(">"):
            continue
        print(line.strip())
```

이 코드는 헤더 줄을 무시하고 실제 서열만 출력합니다. `strip()`은 줄 끝의 줄바꿈 문자를 제거하는 역할을 합니다.

## 예제 실습: FASTA 파일에서 순수한 서열만 추출하기

이제 FASTA 파일에서 서열만 모아 하나의 문자열로 만드는 함수를 작성해 보겠습니다.

```python
def parse_fasta(file_name):
    sequence = ""

    with open(file_name, "r") as f:
        for line in f:
            if line.startswith(">"):
                continue
            sequence += line.strip()

    return sequence


raw_seq = parse_fasta("sample.fasta")
print("읽어온 서열:", raw_seq)
print("서열 길이:", len(raw_seq))
```

이 함수는 FASTA 파일을 열고, 헤더 줄은 건너뛰고, 서열 줄만 이어 붙여 하나의 긴 문자열로 반환합니다. 이후에는 이 문자열을 이용해 길이 계산, GC 함량 계산, 염기 빈도 분석 등 다양한 작업을 이어서 할 수 있습니다.

예를 들어 이전 장에서 만든 GC 함량 함수를 연결할 수도 있습니다.

```python
def calculate_gc_content(dna_seq):
    gc_count = dna_seq.count("G") + dna_seq.count("C")
    return (gc_count / len(dna_seq)) * 100


raw_seq = parse_fasta("sample.fasta")
gc = calculate_gc_content(raw_seq)
print(f"GC 함량: {gc:.2f}%")
```

이처럼 파일 읽기와 서열 분석 함수가 결합되면, 비로소 "실제 데이터 분석"다운 흐름이 만들어집니다.

## 핵심 정리

- 실제 생물정보학 데이터는 대부분 파일 형태로 저장된다.
- `open()` 함수로 텍스트 파일을 읽고 쓸 수 있다.
- FASTA 파일은 헤더 줄과 서열 줄로 구성된다.
- `>`로 시작하는 헤더 줄을 제외하면 실제 서열 데이터만 추출할 수 있다.

# 4장 함수로 만드는 작은 생물정보학 도구

3장에서 우리는 조건문과 반복문을 이용해 DNA 서열을 검사하는 방법을 배웠습니다. 그런데 같은 계산을 여러 번 해야 할 때마다 코드를 길게 다시 써야 한다면 불편할 것입니다. 예를 들어 서열의 GC 함량을 여러 번 계산하거나, DNA를 RNA로 바꾸는 작업을 반복해야 한다면 코드를 매번 복사해 붙여 넣는 방식은 비효율적입니다.

이 문제를 해결하는 도구가 함수(function)입니다. 함수는 자주 사용하는 작업을 하나의 이름으로 묶어 두는 장치입니다. 생물정보학에서는 특정 계산을 반복해서 수행하는 경우가 많기 때문에, 함수는 코드를 더 간결하고 읽기 쉽게 만들 뿐 아니라 실수를 줄이는 데도 도움이 됩니다.

![그림 4-1. 함수가 생물정보학 분석을 재사용 가능한 도구로 바꾸는 과정](figures/chapter04_functions.svg)

그림 4-1은 이 장의 핵심을 보여 줍니다. 서열 분석 과정을 함수로 묶으면, 같은 작업을 여러 DNA 서열에 대해 반복 적용할 수 있고 코드도 훨씬 구조적으로 정리됩니다.

## 학습 목표

- 함수의 기본 개념을 설명할 수 있다.
- 입력값과 반환값의 의미를 이해할 수 있다.
- GC 함량을 계산하는 함수를 작성할 수 있다.
- DNA를 RNA로 바꾸는 전사 함수를 작성할 수 있다.

## 함수가 왜 필요한가?

함수는 "특정 작업을 수행하는 작은 기계"라고 생각하면 이해하기 쉽습니다. 버튼을 누르면 항상 같은 방식으로 작동하는 기계처럼, 함수도 이름을 호출하면 정해진 작업을 수행합니다.

예를 들어 어떤 DNA 서열의 GC 함량을 계산하는 코드를 여러 번 사용해야 한다면, 그 계산 과정을 함수로 묶어 둘 수 있습니다. 그러면 이후에는 긴 코드를 반복해서 쓰지 않고, 함수 이름만 호출하면 됩니다. 이는 코드의 재사용성을 높이고, 프로그램을 더 읽기 쉽게 만듭니다.

## 입력값과 반환값의 개념

함수는 보통 입력값을 받아 작업을 수행한 뒤 결과를 돌려줍니다. 입력값은 함수가 처리할 재료이고, 반환값은 처리 결과라고 볼 수 있습니다.

다음은 아주 간단한 함수 예제입니다.

```python
def say_hello():
    print("안녕하세요")
```

이 함수는 입력값도 없고 반환값도 없지만, 호출하면 정해진 문장을 출력합니다.

입력값과 반환값이 있는 함수는 다음과 같이 만들 수 있습니다.

```python
def get_length(sequence):
    return len(sequence)
```

이 함수는 `sequence`라는 입력값을 받아 그 길이를 반환합니다.

```python
dna = "ATGCGTAC"
print(get_length(dna))
```

## GC 함량 계산 함수 만들기

생물정보학에서 GC 함량은 매우 자주 등장하는 개념입니다. GC 함량은 DNA 서열에서 G와 C가 차지하는 비율을 의미합니다. 일반적으로 다음과 같이 계산합니다.

GC% = (G의 개수 + C의 개수) / 전체 서열 길이 × 100

이 계산을 함수로 만들면 다음과 같습니다.

```python
def calculate_gc_content(dna_seq):
    dna_seq = dna_seq.upper()
    gc_count = dna_seq.count("G") + dna_seq.count("C")
    return (gc_count / len(dna_seq)) * 100
```

함수를 호출해 보겠습니다.

```python
my_dna = "ATGCGTACGTTAG"
gc = calculate_gc_content(my_dna)
print(f"GC 함량: {gc:.2f}%")
```

이 함수는 입력된 서열을 대문자로 바꾼 뒤, G와 C의 개수를 세고 전체 길이로 나누어 백분율을 계산합니다. 이러한 방식은 생물정보학에서 아주 기본적이면서도 중요한 분석 방법입니다.

## DNA를 RNA로 바꾸는 전사 시뮬레이션

생물학의 중심원리에 따르면 DNA의 정보는 RNA로 전사(transcription)될 수 있습니다. 파이썬에서는 DNA 문자열에서 `T`를 `U`로 바꾸어 간단한 전사 과정을 모의할 수 있습니다.

```python
def transcribe(dna_seq):
    return dna_seq.upper().replace("T", "U")
```

함수를 사용하면 다음과 같습니다.

```python
dna = "ATGCGTAC"
rna = transcribe(dna)

print("DNA:", dna)
print("RNA:", rna)
```

이 함수는 DNA를 실제 생물학적 반응처럼 완벽하게 재현하는 것은 아니지만, 생물학 개념을 코드로 바꾸는 좋은 연습이 됩니다.

## 예제 실습: 나만의 서열 분석 함수 만들기

지금까지 배운 내용을 바탕으로 DNA 서열의 길이와 GC 함량을 함께 알려 주는 함수를 만들어 보겠습니다.

```python
def analyze_dna(dna_seq):
    dna_seq = dna_seq.upper()
    length = len(dna_seq)
    gc_count = dna_seq.count("G") + dna_seq.count("C")
    gc_ratio = (gc_count / length) * 100

    print("DNA 서열:", dna_seq)
    print("서열 길이:", length)
    print(f"GC 함량: {gc_ratio:.2f}%")


analyze_dna("ATGCGTACGTTAG")
```

이 함수는 입력된 서열을 받아 길이와 GC 함량을 한 번에 출력합니다. 비록 간단한 예제이지만, 이미 작은 생물정보학 도구의 형태를 갖추고 있습니다.

## 핵심 정리

- 함수는 자주 사용하는 작업을 묶어 재사용 가능하게 만든다.
- 입력값은 함수가 처리할 데이터이고, 반환값은 함수가 돌려주는 결과이다.
- GC 함량 계산은 함수로 만들기에 좋은 대표적인 생물정보학 작업이다.
- DNA를 RNA로 바꾸는 전사 시뮬레이션은 생물학 개념을 코드로 연결하는 좋은 예이다.

# 8장 NumPy로 시작하는 수치 데이터 분석

생물정보학은 DNA 서열만 다루는 학문이 아닙니다. 실제 연구 현장에서는 유전자 발현량, 샘플 간 거리, 단백질의 물리적 특성, 실험 반복 측정값처럼 수많은 숫자 데이터를 함께 분석합니다. 서열이 생명의 문자라면, 수치 데이터는 생명 현상을 정량적으로 보여 주는 숫자의 언어라고 할 수 있습니다.

이 장에서는 파이썬의 대표적인 수치 계산 라이브러리인 NumPy를 처음 소개합니다. NumPy는 많은 숫자를 빠르고 효율적으로 다룰 수 있게 해 주며, 이후에 배울 Pandas와 시각화 도구의 기반이 되기도 합니다. 즉, NumPy는 생물정보학에서 "숫자로 된 데이터"를 본격적으로 다루기 위한 첫 번째 도구입니다.

![그림 8-1. NumPy 배열이 생물정보학 수치 데이터를 다루는 방식](figures/chapter08_numpy.svg)

그림 8-1은 이 장의 핵심을 보여 줍니다. 여러 샘플의 발현값이나 측정값을 NumPy 배열에 담으면, 반복 계산과 통계 처리를 더 간단하고 체계적으로 수행할 수 있습니다.

## 학습 목표

- 생물정보학에서 수치 데이터가 중요한 이유를 설명할 수 있다.
- 리스트와 배열의 차이를 이해할 수 있다.
- NumPy 배열을 만들고 기본 연산을 수행할 수 있다.
- 평균과 합계 같은 간단한 통계량을 계산할 수 있다.

## 생물정보학에서 수치 데이터가 중요한 이유

생물정보학은 서열 분석뿐 아니라 발현 데이터, 빈도 데이터, 거리 계산, 확률 계산 등 다양한 수치 작업을 포함합니다. 예를 들어 RNA 발현량 실험에서는 유전자마다 숫자값이 측정되고, 여러 샘플을 비교하면서 평균이나 차이를 계산해야 합니다.

만약 이런 숫자를 모두 일반 리스트로만 다루려 한다면 불편한 점이 많습니다. 리스트도 숫자를 담을 수는 있지만, 대량의 수치 데이터를 빠르고 편리하게 계산하기에는 제한이 있습니다. 이 지점에서 NumPy 배열이 큰 도움을 줍니다.

## 배열과 리스트의 차이

리스트는 파이썬의 기본 자료구조이고, NumPy 배열은 수치 계산에 특화된 자료구조입니다. 겉보기에는 비슷해 보이지만, 배열은 숫자 데이터를 같은 형식으로 정리해 빠르게 계산하는 데 강점을 가집니다.

먼저 리스트를 살펴보겠습니다.

```python
expression_list = [12, 15, 18, 20, 17]
print(expression_list)
```

NumPy 배열은 다음과 같이 만듭니다.

```python
import numpy as np

expression_array = np.array([12, 15, 18, 20, 17])
print(expression_array)
```

두 결과는 비슷해 보이지만, 배열은 이후 수학 연산에서 훨씬 유용합니다.

## 유전자 발현 데이터를 배열로 표현하기

예를 들어 다섯 개 샘플의 발현값이 있다고 가정해 보겠습니다.

```python
import numpy as np

expression = np.array([12, 15, 18, 20, 17])
print("발현 데이터:", expression)
```

NumPy 배열은 여러 값을 하나의 묶음으로 저장하면서도, 각 값에 대해 수학 연산을 쉽게 수행할 수 있습니다.

```python
print("모든 값에 2를 곱한 결과:", expression * 2)
```

이 코드는 배열의 각 원소에 2를 곱한 새 결과를 반환합니다. 리스트에서도 비슷한 작업을 하려면 반복문을 직접 써야 하지만, NumPy 배열에서는 훨씬 간단합니다.

## 기본 통계량 계산하기

생물정보학에서는 데이터를 읽는 것만큼이나 요약하는 일이 중요합니다. 평균, 합계, 최댓값, 최솟값 같은 기본 통계량은 데이터의 전체적인 성격을 빠르게 파악하게 해 줍니다.

```python
import numpy as np

expression = np.array([12, 15, 18, 20, 17])

print("합계:", np.sum(expression))
print("평균:", np.mean(expression))
print("최댓값:", np.max(expression))
print("최솟값:", np.min(expression))
```

이러한 계산은 유전자 발현 데이터나 실험 측정값을 이해하는 데 매우 기본적이면서도 중요합니다.

## 예제 실습: 간단한 발현 데이터 분석

이제 NumPy를 이용해 두 샘플 집단의 발현 데이터를 비교하는 간단한 예제를 살펴보겠습니다.

```python
import numpy as np

control = np.array([10, 12, 11, 13, 12])
treated = np.array([15, 18, 17, 16, 19])

print("대조군 평균:", np.mean(control))
print("처리군 평균:", np.mean(treated))
print("평균 차이:", np.mean(treated) - np.mean(control))
```

이 코드는 대조군과 처리군의 평균 발현값을 비교합니다. 실제 연구에서는 이보다 훨씬 많은 샘플과 복잡한 통계가 사용되지만, 핵심 아이디어는 같습니다. 숫자 데이터를 구조화하고, 요약하고, 비교하는 것입니다.

## 핵심 정리

- 생물정보학에서는 서열뿐 아니라 다양한 수치 데이터도 중요하다.
- NumPy 배열은 수치 계산에 특화된 자료구조이다.
- 배열은 리스트보다 대량의 숫자 데이터를 다루기에 편리하다.
- `np.sum()`, `np.mean()`, `np.max()`, `np.min()` 같은 함수로 기본 통계량을 계산할 수 있다.

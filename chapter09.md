# 9장 Pandas로 유전자 데이터 정리하기

8장에서 우리는 NumPy를 이용해 수치 데이터를 배열로 다루는 법을 배웠습니다. 하지만 실제 생물정보학 데이터는 숫자만 덩어리처럼 모여 있지 않습니다. 보통은 유전자 이름, 샘플 이름, 발현량, 그룹 정보처럼 여러 종류의 정보가 표 형태로 함께 정리되어 있습니다. 즉, 실제 데이터 분석에서는 "표를 읽고 다루는 능력"이 매우 중요합니다.

이 장에서는 Pandas를 사용해 표 형태의 생물정보학 데이터를 다루는 방법을 배웁니다. Pandas의 핵심 자료구조인 데이터프레임(DataFrame)은 엑셀 표와 비슷하게 행과 열로 데이터를 관리할 수 있게 해 줍니다. 따라서 유전자 발현 데이터, 샘플 메타데이터, 통계 결과 표 등을 정리하는 데 매우 유용합니다.

그림 9-1은 이 장의 핵심을 보여 줍니다. 유전자 이름과 발현값을 행과 열로 정리하면, 특정 열을 선택하거나 조건에 맞는 행만 쉽게 골라낼 수 있습니다.

## 학습 목표

- 데이터프레임(DataFrame)이 무엇인지 설명할 수 있다.
- CSV 파일을 Pandas로 읽어 올 수 있다.
- 특정 열을 선택하고 조건에 맞는 행을 필터링할 수 있다.
- 간단한 유전자 발현표를 정리할 수 있다.

## 데이터프레임(DataFrame) 이해하기

데이터프레임은 표 형태의 데이터를 다루기 위한 Pandas의 대표 구조입니다. 각 열은 하나의 속성을 나타내고, 각 행은 하나의 관측값을 나타냅니다.

예를 들어 다음과 같은 표를 생각해 볼 수 있습니다.

| Gene | Expression |
|------|------------|
| TP53 | 12.5 |
| BRCA1 | 8.7 |
| EGFR | 15.2 |

이러한 구조는 생물정보학에서 매우 흔합니다. 유전자 이름과 발현값, 샘플 이름과 그룹 정보, 단백질 이름과 기능 설명 등 다양한 정보를 이런 표로 정리합니다.

## CSV 파일 읽기

Pandas를 사용하면 CSV 파일을 매우 쉽게 읽어 올 수 있습니다.

```python
import pandas as pd

df = pd.read_csv("gene_expression.csv")
print(df)
```

이 코드는 `gene_expression.csv`라는 파일을 읽어 데이터프레임으로 저장합니다. 이후에는 이 `df`를 이용해 다양한 분석을 수행할 수 있습니다.

만약 파일 없이 바로 예제를 만들고 싶다면, 다음처럼 딕셔너리로 데이터프레임을 만들 수도 있습니다.

```python
import pandas as pd

data = {
    "Gene": ["TP53", "BRCA1", "EGFR", "MYC"],
    "Expression": [12.5, 8.7, 15.2, 19.1]
}

df = pd.DataFrame(data)
print(df)
```

## 열 선택과 행 필터링

데이터프레임의 장점은 필요한 열만 쉽게 선택할 수 있다는 점입니다.

```python
print(df["Gene"])
print(df["Expression"])
```

또한 조건을 사용해 특정 값 이상인 행만 골라낼 수도 있습니다.

```python
high_expr = df[df["Expression"] > 10]
print(high_expr)
```

이 코드는 발현값이 10보다 큰 유전자만 추출합니다. 생물정보학에서는 특정 기준 이상으로 발현된 유전자, 특정 조건을 만족하는 샘플, 특정 분류군에 속한 데이터만 골라내는 일이 매우 흔합니다.

## 특정 발현값 이상의 유전자 찾기

이제 조금 더 실용적인 예를 보겠습니다.

```python
import pandas as pd

data = {
    "Gene": ["TP53", "BRCA1", "EGFR", "MYC", "PTEN"],
    "Expression": [12.5, 8.7, 15.2, 19.1, 7.9]
}

df = pd.DataFrame(data)

filtered = df[df["Expression"] >= 10]
print(filtered)
```

이 결과는 발현값이 10 이상인 유전자만 보여 줍니다. 이런 식의 필터링은 데이터 분석에서 가장 자주 사용하는 작업 중 하나입니다.

## 예제 실습: 유전자 발현표 정리하기

이제 작은 유전자 발현표를 만들고, 정렬과 필터링을 함께 수행해 보겠습니다.

```python
import pandas as pd

data = {
    "Gene": ["TP53", "BRCA1", "EGFR", "MYC", "PTEN"],
    "Expression": [12.5, 8.7, 15.2, 19.1, 7.9]
}

df = pd.DataFrame(data)

print("원본 데이터")
print(df)

print("\n발현값이 10 이상인 유전자")
print(df[df["Expression"] >= 10])

print("\n발현값 기준 내림차순 정렬")
print(df.sort_values(by="Expression", ascending=False))
```

이 코드는 세 가지 중요한 작업을 보여 줍니다.

1. 데이터프레임 생성
2. 조건 필터링
3. 값 기준 정렬

이 세 가지는 실제 데이터 분석에서도 매우 자주 사용되는 기본 작업입니다.

## 핵심 정리

- Pandas는 표 형태의 데이터를 다루는 데 매우 유용한 라이브러리이다.
- 데이터프레임은 행과 열로 구성된 표 구조이다.
- `read_csv()`를 사용하면 CSV 파일을 쉽게 읽어 올 수 있다.
- 특정 열 선택, 조건 필터링, 정렬은 데이터프레임 분석의 기본 작업이다.
